Développement d'un jeu en Python

Contexte :


Durant mon année de Terminal, nous devions réaliser un projet dans le cadre de ma Spécialité NSI. Le projet était de réaliser un jeu en python par groupe de 4.

.

Objectif :


L'objectif était de créer un jeu en Python à partir du module PyGame. Le choix du jeu étant libre, nous avons décidé de faire un Shoot'Em Up.

.

Travail réalisé


J'ai été le chef de projet lors de la réalisation de notre jeu. J'ai donc dû répartir les tâches entre nous.

Tout ce dont je vais parler a été réalisé en programmation orientée objet.

Personnellement, je me suis occupé de l'interaction entre le joueur et le vaisseau dans le jeu. J'ai donc dû définir comment le vaisseau pouvait se déplacer et tirer, les touches du clavier que Python devait interpréter. Cette partie était complexe mais j'ai eu une immense satisfaction lorsque je l'ai réussie.

J'ai ensuite dû créer des ennemis. Pour les ennemis, les déplacements devaient être aléatoires. J'ai donc utilisé le module random afin de pouvoir créer ce déplacement aléatoire. Il fallait également que les ennemis puissent tirer sur le joueur.

Une fois que nous avions un vaisseau et des ennemis, nous devions gérer les collisions entre les tirs et le joueur ou un ennemi. Pour cela nous avons dû stocker la position de nos tirs dans des listes de listes afin de pouvoir réutiliser la position et faire une fonction la détectant.

Nous avons ensuite implémenté une fonction de vie pour le vaisseau du joueur et afficher un menu d'échec si le joueur perd toute sa vie.

.

Résultat :


La réalisation de notre projet en Python m'a permis de développer mes compétences en Python ainsi que mon imagination car nous étions libres sur le jeu à réaliser et comment le faire.

J'ai également acquis des compétences en pilotage projet, notamment dans le définition et le partage des tâches.

.
